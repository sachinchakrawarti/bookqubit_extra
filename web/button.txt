============================================
           BUTTON ACTION TYPES
============================================

| Type      | Description              | Example Usage                        |
|-----------|--------------------------|--------------------------------------|
| onClick   | Client-side function     | onClick={() => handleClick()}        |
| to        | Internal navigation      | to="/books/123"                      |
| href      | External link            | href="https://example.com"           |
| onToggle  | Toggle state             | onToggle={() => setLiked(!liked)}    |
| onShare   | Share content            | onShare={() => shareContent()}       |
| onReport  | Report content           | onReport={() => reportContent()}     |
